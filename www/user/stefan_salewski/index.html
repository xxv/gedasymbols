All my symbols and footprints are licensed GPL for distribution, and unlimited for use. They are created with care, but most are still untested on real hardware.

Larger rectangular symbols are created with the Python script Tragesym.

For a few standard symbols (capacitor, resistor) I have created horizontal and vertical versions with aligned text, and special versions with short pins. In general case short pins look ugly and should not be used, but in a few cases they can be useful.

Some symbols uses footprints of John Luciani's footprint page or my own footprints (see below). Most of my footprints are generated with my own command line based Ruby script SFG. (For some footprints a special version optimized for hand soldering with name extension "_HS" exists.)
